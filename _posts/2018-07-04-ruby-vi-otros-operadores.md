# Booleanos

En Ruby, existen dos valores Booleanos: **true** (verdadero) y *false* (falso):

{% highlight ruby %}
isOnline = true
userIsAdmin = false
{% endhighlight %}

**Nota:** otro valor que encontrarás usualmente es `nil`, el cual representa la ausencia de valor (vacío).
